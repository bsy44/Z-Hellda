# ⚔️ Zhellda 

Un Zelda-like en 2D se déroulant dans un monde apocalyptique plongé dans une dimension infernale, où des créatures corrompues ont remplacé toute forme de vie.

## 🎮 Gameplay

### 🕹️ Déplacements & contrôles

Le joueur itéragit et se déplace librement dans l’environnement avec des contrôles directionnels simples :
- **Déplacement** dans les quatre directions
- **Attaques directionnelles** (haut, bas, gauche, droite)
- **Interactions** avec l’environnement (coffres, dialogues, objets)

## 👾 Ennemis

Plusieurs types d’ennemis viennent rythmer la progression :

- **Ennemis résistants** mais lents
- **Ennemis rapides** et mais avec peu de vie
- **Entités** plus puissantes nécessitant prudence
- Un **boss final** demandant une maîtrise complète du gameplay

## 🗡️ Armes

Différents équipements sont disponibles :

- **Épée** : équilibrée, courte portée
- **Arc** : attaques à distance
- **Marteau** : dégâts de zone

Chaque arme offre une approche différente du combat.

## 🎒 Objets & inventaire

Le joueur dispose d’un inventaire dynamique :

- **Objets de soin**
- **Bonus temporaires**
- **Améliorations de combat**

Le choix des objets influence directement la survie.
